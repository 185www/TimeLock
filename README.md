## 📱 这是什么？

参考何同学「时间锁」理念，做一个**真正好用**的防沉迷工具。

你设定时间，手机在这段时间内彻底变为「砖头」，让你强制专注于现实世界。

## ✨ 核心特性

- **🔒 强制锁定** — 利用 Android Screen Pinning + Device Admin，无法绕过
- **📋 白名单** — 保留紧急通话和必要 App
- **🛡️ 防绕过** — 重启恢复、防 ADB、防卸载
- **📊 专注统计** — 记录你的每一次专注
- **🧩 番茄钟** — 内置 25+5 标准工作法
- **🔓 完全开源** — 代码可审计，无后门，不上传任何数据
